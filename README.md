## Invoice Generator

### Live URL: [http://invoicer-eta.vercel.app/](https://invoice-mauve.vercel.app/)

![Web Preview](./src/images/githubSnapshot.png)

### Tools Used:

- React.js
- Firebase
- React-to-print
- Tailwind CSS
- Material UI
- React Hook Form
- React Toastify
- Redux Toolkit

### To work on this project

- Clone the repository
- Run `npm install`
- Start the development `npm start`
